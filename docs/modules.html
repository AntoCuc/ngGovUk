Angular Modules

Some angular modules that can be included in any project for some global gds designs

global-nav

To include the header add the directive by using the data attribute data-global-nav in the mark up.

In the javascript create an object called globalNav which can contain two objects: pageTitle and navItems. If your page does not require either of those, they can simply be omitted.

angular.module('app').controller('globalNavDemoCtrl', function ($scope) { $scope.globalNav = { pageTitle: { title: 'Criminal Justice System', type: 'text', ref: '#' }, navItems: [ { title: 'Getting Started', type: 'href', ref: '#!/modules' }, { title: 'CSS Elements', type: 'href', ref: '#!/modules' }, { title: 'UI Components', type: 'href', ref: '#!/modules' }, { title: 'Ng Modules', type: 'href', ref: '#!/modules' }, { title: 'API', type: 'href', ref: '#!/modules' } ] }; });
